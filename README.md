## **🛠 구현 기능**
- **검색**<br>
: 검색을 통해 사용자가 원하는 정보 제공
- **이미지 삽입**<br>
: 유기동물 이미지 삽입해 어떤 동물인지 보여줌
- **지도**<br>
: 보호소의 위치를 파악할 수 있도록 지도 활용
- **카테고리별 정렬**<br>
: 지역, 품종으로 카테고리를 나눠 보기 편하게 정렬
- **이메일**<br>
: 사용자가 원하는 정보 이메일로 제공
- **그래프**<br>
: 지역, 품종으로 나눠 그래프를 그려 전체적인 정보를 보기 쉽게 함
- **C/C++ 연동**<br>
: C/C++ 연동
- **텔레그램 봇**<br>
: 텔레그램 봇과 연동
